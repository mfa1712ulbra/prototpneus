# **App Name**: TireControl

## Core Features:

- Autenticação de Usuário: Login e autenticação seguros para gerentes e técnicos.
- Registro de Veículos: Registrar e consultar detalhes dos veículos. Listagem e seleção de caminhões registrados.
- Diagrama do Veículo: Diagrama visual do veículo, exibindo a posição de cada pneu no caminhão.
- Informações do Pneu: Registrar nova pressão dos pneus, profundidade do piso, observações e movimentos.
- Controle de Movimentação: Registrar mudanças de status, como rotação, recapagem, armazenamento, reparo ou descarte.
- Confirmação de Operação: Mensagens de feedback visual para o usuário após salvar ou cancelar operações.
- Relatórios: Consultar o histórico de movimentações, recapagens e descartes. Isso inclui armazenar o histórico de manutenções realizadas e gerar relatórios.

## Style Guidelines:

- Cor primária: Verde-azulado escuro (#008080) para uma sensação profissional e confiável.
- Cor de fundo: Cinza claro (#F0F0F0) para fornecer um pano de fundo neutro e melhorar a legibilidade.
- Cor de destaque: Laranja (#FFA500) para chamadas para ação e destaques.
- Fonte do corpo e do título: 'Inter' (sans-serif) para uma aparência limpa e moderna.
- Use ícones claros e funcionais para peças de veículos e indicadores de status.
- Layout intuitivo e claro com fluxos rápidos para garantir fácil entendimento. Totalmente desenvolvido para mobile.
- Animações sutis para confirmações de operação e atualizações de dados.
